## 🦴 What Is This Script?

This repository is the most actively maintained and feature-rich **roblox ro ghoul script** collection available in 2026. Built specifically for **Ro-Ghoul** — one of Roblox's most popular anime-inspired combat RPGs based on Tokyo Ghoul — this script suite automates every grind-heavy mechanic and adds powerful combat tools that give you a decisive edge in every encounter.

**Ro-Ghoul** demands enormous amounts of yen and RC cells to unlock the best kagune, quinque, and upgrades. Farming manually is a slow, repetitive process that can take weeks. A well-configured `ro ghoul script auto farm` loop handles all of that passively — grinding mobs, farming bosses, and stacking yen continuously whether you're actively playing or fully AFK.

Beyond farming, this script also includes the popular `ro ghoul silent aim script` and `ro ghoul auto aim script` modules for competitive PvP, as well as the beloved `ro ghoul dash boost script` for movement optimization. If you've heard of the community-famous **Porry Script** for Ro-Ghoul, you'll find that the `porry script ro ghoul` loader is fully supported in this repository as an alternative entry point.

## 🐛 Common Errors & Fixes

### ❌ Auto Farm Gyakusatsu not teleporting correctly

**Cause:** Zone coordinates may have shifted after a map update.
**Fix:** Pull the latest build from the [Releases](https://rogoulsc.github.io/ro-ghoul/) page. The `ro ghoul script auto farm gyakusatsu` waypoints are stored in [`/data/zones.json`](https://rogoulsc.github.io/ro-ghoul/) and updated with every patch. You can also manually set coordinates in the **Farm** tab if you know the current zone position.

***

### ❌ Auto Farm Boss missing spawns

**Cause:** The boss respawn timer changed after a game balance update.
**Fix:** Open the **Farm** tab and reset the boss timer manually. The `ro ghoul script auto farm boss` uses a cached timer value — updating to the latest build from [Releases](https://rogoulsc.github.io/ro-ghoul/) will sync it correctly.

***

### ❌ Silent Aim not connecting hits

**Cause:** The `ro ghoul silent aim script` target reference becomes nil if the enemy moves out of range between the aim lock and the attack execution.
**Fix:** Reduce your movement speed slightly and increase the aim lock range in the **Combat** tab. Alternatively, switch to `AutoAim` mode which is more stable for most combat scenarios.

***

### ❌ Dash Boost causing kick

**Cause:** A `DashMultiplier` value above `3.0` can trigger Roblox's speed detection in some servers.
**Fix:** Lower `DashMultiplier` to `2.0` or `2.5` in [`config.json`](https://rogoulsc.github.io/ro-ghoul/). The `ro ghoul dash boost script` is most stable between `1.5` and `2.5` — values above `3.0` carry a higher kick risk on public servers.

***

### ❌ Inf Yen not accumulating visibly

**Cause:** The yen counter UI may not refresh in real time even though the balance is increasing.
**Fix:** Open your in-game stats menu to force a UI refresh. The `ro ghoul script inf yen` module operates at the session level and the balance updates correctly — it's a display lag issue only.

***

### ❌ Porry Script loader not connecting

**Cause:** The `porry script ro ghoul` remote endpoint may have changed.
**Fix:** Use the updated loader from [`/loaders/porry.lua`](https://rogoulsc.github.io/ro-ghoul/) in this repository. It's kept in sync with the latest Porry Script endpoint. Alternatively, switch to the native [`loader.lua`](https://rogoulsc.github.io/ro-ghoul/) for identical functionality.

***

### ❌ "loadstring is not enabled" error

**Cause:** Your executor has loadstring disabled by default.
**Fix:** Enable it in executor settings. Delta Executor and Solara both have it active out of the box, making them the easiest entry point for first-time users.

## ❓ FAQ

**Q: Does the Porry Script loader work with this repository?**
Yes — the `porry script ro ghoul` loader in [`/loaders/porry.lua`](https://rogoulsc.github.io/ro-ghoul/) is maintained alongside the main script and loads all the same modules through the Porry Script interface.

**Q: Which farm zone gives the most yen?**
The `ro ghoul script auto farm gyakusatsu` loop consistently produces the highest yen-per-hour of any available zone. Set `FarmZone` to `"gyakusatsu"` in your config for maximum efficiency.

**Q: Is silent aim safe to use in PvP?**
The `ro ghoul silent aim script` is designed to be discreet, but no module is 100% undetectable. Use it on an alternate account and avoid obvious behavior like hitting targets from extreme distances.

**Q: Does the auto aim work for both ghouls and CCG?**
Yes. The `ro ghoul auto aim script` works for both factions and automatically adjusts target priority based on your current faction alignment.

**Q: Can I stack dash boost with speed modifier?**
Yes — the `ro ghoul dash boost script` and speed modifier are independent and stack additively. Keep the combined effect below detection thresholds by staying under `DashMultiplier: 2.5` and `SpeedMultiplier: 2.0` simultaneously.

**Q: Does auto farm boss work for all bosses?**
The `ro ghoul script auto farm boss` module supports all currently spawnable bosses in the game. The boss roster is stored in [`/data/bosses.json`](https://rogoulsc.github.io/ro-ghoul/) and updated with every patch.
